Sistema i18n - Demo

Prueba los diferentes atributos y funcionalidades del sistema de internacionalización.

1. Text simple (data-i18n)

Para traducir texto simple, usa el atributo data-i18n con la clave:

<p data-i18n="hero.eyebrow">Texto original</p>
Transferencia tecnológica · Ingeniería aplicada

2. HTML con etiquetas (data-i18n-html)

Si la traducción tiene HTML, usa data-i18n-html:

<h1 data-i18n-html="hero.title">...</h1>
Transferencia tecnológica
e ingeniería aplicada.

3. Traducción de múltiples elementos

Todos los elementos con data-i18n se actualizan automáticamente:

<a href="#" data-i18n="nav.cta">...</a>

4. Listas dinámicas (data-i18n-list)

Para generar <li> items desde un array:

<ul data-i18n-list="capacidades.cards.electronica.items"></ul>
  • Item 1

5. Obtener traducción desde JavaScript

Usa la función i18n.t(clave) para obtener traducciones dinámicamente:

const title = i18n.t('hero.title');
const subtitle = i18n.t('capacidades.title');
// O en otro idioma específico:
const titleEn = i18n.t('hero.title', 'en');
